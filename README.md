# Blinkit Sales Analytics Dashboard With blinkit website theme::
<img width="1411" alt="Screenshot 2567-07-21 at 1 30 42 AM" src="https://github.com/user-attachments/assets/7f72b1db-20f5-4a32-b847-d76e39a4c03d">
This repository contains a detailed sales analytics dashboard created for Blinkit, a hyper-local delivery app. Developed using Power BI, the dashboard leverages DAX functions to provide insightful analysis of sales data, outlet performance, and customer ratings across various locations.

### Features
- Total Sales: Visualize total sales figures with a comprehensive overview.
- Average Sales: Calculate and display the average sales value.
- Item Analysis: Track the number of total items sold and their performance.
- Customer Ratings: Analyze average order ratings to understand customer satisfaction.
- Trend Analysis: Observe sales trends over the years and identify key patterns.
- Outlet Performance: Evaluate sales performance based on outlet location type and size.
- Product Category Insights: Identify top-performing categories such as fruits, snacks, and household items.

### Data Attributes:
- year: Year of data collection
- customer_id: Unique identifier for each customer
- Item Fat Content: The fat content category of the items.
- Item Identifier: Unique identifier for each item.
- Item Type: Category of the item.
- Outlet Establishment Year: Year when the outlet was established.
- Outlet Identifier: Unique identifier for each outlet.
- Outlet Location Type: Type of location for the outlet (e.g., Tier 1, Tier 2, etc.).
- Outlet Size: Size of the outlet (e.g., Small, Medium, Large).
- Outlet Type: Type of outlet (e.g., Supermarket, Grocery Store, etc.).
- Item Visibility: Visibility of the item in the store.
- Item Weight: Weight of the item.
- Sales: Sales figures for the items.
- Rating: Customer ratings for the items and outlets.
